## 项目描述

一款智能收纳管理应用，帮助用户整理物品，支持语音识别和搜索查找。

## 功能特性

- 物品管理 - 添加、编辑、删除物品
- 分类整理 - 按类别整理物品
- 语音识别 - 语音录入物品信息
- 搜索查找 - 快速搜索物品
- 统计分析 - 统计物品数量和类别

## 技术栈

- React
- TypeScript
- Tailwind CSS
- Vite
- Zustand

## 快速开始

直接打开 index.html 文件即可使用。
